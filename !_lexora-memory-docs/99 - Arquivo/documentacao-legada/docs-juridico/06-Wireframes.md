# Wireframes Textuais das Telas Principais

## 1. Home Operacional (ADV, EST, COO)
- Cabeçalho: Meu Dia + ações rápidas (novo atendimento, nova tarefa, novo documento, ver agenda)
- Cards: prazos hoje, tarefas pendentes, processos aguardando ação, clientes aguardando retorno
- Núcleo: fila de responsabilidades (processo, cliente, prazo, status, ação)
- Painel lateral: agenda de hoje, últimas movimentações, alertas críticos
- Analítico básico: processos por fase, tarefas por status
- Rodapé: documentos faltantes, publicações recentes

## 2. Home Administrativa (ADM, COO, SOC)
- Cabeçalho: Visão Geral do Escritório + ações (distribuir demanda, criar tarefa, ver relatórios)
- Cards executivos: processos ativos, sem atualização, prazos críticos, tarefas atrasadas, equipe sobrecarregada
- Mapa operacional: equipes/responsáveis com KPI
- Analítico: gráficos por área, produtividade por responsável
- Tabela de gestão rápida: processos parados, pendências, gargalos

## 3. Home Financeira (FIN, SOC)
- Cabeçalho: Financeiro + botões (nova cobrança, registrar pagamento, emitir nota)
- Cards: recebimentos hoje, inadimplentes, contas a pagar, saldo projetado
- Lista de cobranças com filtros e ações rápidas
- Painel lateral: fluxo de caixa, inadimplência, contratos vencendo
- Gráficos: recebimentos por período, inadimplência por faixa

## 4. Processo (coluna, detalhes, timeline)
- Topo: header do processo + ação rápida (andamento, prazo, documento, peça)
- Cards de resumo: fase atual, próximo prazo, audiência futura, documentos pendentes
- Conteúdo: 3 colunas (resumo, timeline, tarefas/documents)
- Abas funcionais: visão geral, andamentos, prazos, audiências, documentos, tarefas, publicações, atendimento, financeiro
- Drawer contextual ao clicar em item de timeline

## 5. Agenda e Prazos
- Cabeçalho: período (dia/semana/mês) + filtros + ações
- Cards: prazos hoje, vencidos, audiências próximas
- Calendário amplo + lista do dia
- Tabela operacional filtrável

## 6. Detalhe do Cliente
- Header: cliente + ações
- Cards: processos ativos, documentos pendentes, saldo, último atendimento
- Abas: visão geral, dados, processos, documentos, atendimentos, financeiro

## 7. Administração
- Abas: usuários, perfis, permissões, equipes, logs, LGPD, dashboards
- Cada perfil exibe permissões/scope/dashboards
- Editor de dashboard por perfil (widgets, ordem, obrigatórios)
