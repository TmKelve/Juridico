# Casos de Uso (User Stories + Job-to-Be-Done)

## Visão geral
Esta seção documenta as principais jornadas do usuário por perfil, com foco em resultados e valor entregues.

## 1. Advogado: Gerenciar processo e prazos
- Story: como advogado, quero ver os prazos do dia e ações de processo, para priorizar tarefas e evitar perdas de prazo.
- Job-to-be-Done: quando tenho vários processos a acompanhar, preciso de um painel que traga só o que depende de mim, para executar com foco.

## 2. Estagiário: Executar tarefa operacional
- Story: como estagiário, quero receber checklist diário e documentos pendentes, para concluir as demandas sem dúvidas.
- JTBD: quando recebo atividades do time, preciso de uma lista clara do que está atribuído, para não perder tempo.

## 3. Financeiro: Controlar fluxo de caixa e inadimplência
- Story: como financeiro, quero ver recebimentos do dia e inadimplentes, para priorizar cobranças urgentes.
- JTBD: quando o fechamento mensal se aproxima, preciso de previsões e alertas, para ajustar a liquidez do escritório.

## 4. Administrador: Monitorar operação do escritório
- Story: como administrador, quero ver gargalos e produtividade em uma só tela, para redistribuir carga rápido.
- JTBD: quando o time está sobrecarregado, preciso de visão transversal para manter operação estável.

## 5. Sócio: Acompanhar performance estratégica
- Story: como sócio, quero ver receita do mês, rentabilidade e risco em painel executivo, para tomar decisões de investimento.
- JTBD: quando faço reunião de diretoria, preciso de indicadores consolidados, para priorizar ações de negócio.

## 6. Atendimento/Comercial: Converter leads em clientes
- Story: como comercial, quero funil de leads e tarefas de follow-up, para aumentar taxa de conversão.
- JTBD: quando há novos contatos, preciso de visibilidade do pipeline, para não perder oportunidades.

## 7. Cliente Externo: Acompanhar o caso
- Story: como cliente, quero ver status do processo e solicitações de documentos, para reduzir pedidos de informação ao escritório.
- JTBD: quando não sei o andamento do meu processo, preciso de transparência, para sentir segurança no serviço.
