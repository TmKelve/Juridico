# Perfis e Experiência por Função

## 1. Estrutura de perfis base
- Sócio / Direção
- Administrador do Escritório
- Coordenador Jurídico
- Advogado
- Estagiário / Apoio Operacional
- Atendimento / Comercial
- Financeiro
- Cliente Externo (Portal)

## 2. O que muda no produto
Perfil + Permissão + Escopo = Experiência correta.

### 2.1. Perfil / Home por papel
1. Advogado / Operacional Jurídico
- Prazos, ações, clientes, documentos, tarefas, audiências.
2. Estagiário / Apoio Operacional
- tarefas delegadas, documentos, protocolos, checklists.
3. Financeiro
- recebimentos, inadimplência, honorários, contratos, caixa.
4. Administrativo / Gestão
- volume por área, produtividade, gargalos, processos sem update.
5. Sócio / Direção
- receita, novos contratos, rentabilidade, equipe sobrecarregada.

## 3. Modelo de perfis + dashboards
- Workspace Operacional
- Workspace Jurídico
- Workspace Financeiro
- Workspace Administrativo
- Workspace Executivo

## 4. Perfis especiais e múltiplos papéis
- Perfil principal + perfis secundários + escopo específico.
- Exemplo: João = Advogado + Coordenador Cível.

## 5. Prioridade do MVP
- perfis: admin, advogado, estagiário, financeiro, atendimento.
- sócio como variação executiva do administrativo.
- cliente externo fase 2.
