# Roles and Permissions
- `platform_admin`: leitura/escrita total, incluindo bloqueio/cancelamento e memberships.
- `platform_billing`: leitura global + ações comerciais (activate/cancel/reactivate); sem gestão ampla de memberships.
- `platform_support`: leitura/auditoria/suporte contextual read-only.
Fronteira plataforma x tenant: plataforma opera governança de conta; tenant opera domínio jurídico interno.
