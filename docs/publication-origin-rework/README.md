# Publication Origin Rework

Documentação aditiva do rework de origem/publicações, alinhada a `contracts/publication-origin-rework.contract.json` e ao padrão dos pacotes em `docs/epic-*`.

## Escopo validado
- Captura bruta e evidência original antes da consolidação operacional.
- Correlação ponta a ponta por `correlationId` entre captura, evento, publicação e ações derivadas.
- Expansão compatível de `GET /publications`, `GET /triage`, `GET /crm/leads` e `GET /crm/opportunities`.
- Novas consultas dedicadas para captura, timeline e ações derivadas.
- Cobertura documental e smoke aditivo sem alterar o orquestrador nesta entrega.
